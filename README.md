
# 🏡 Paradise Valley - Airbnb Clone

A full-stack web application inspired by Airbnb where users can explore, create, edit, and manage beautiful stay listings.

🌐 Live Demo: https://paradise-valley.onrender.com/listing  
💻 GitHub Repo: https://github.com/unlimitedcode07/paradise  

---

## 🚀 Features

- 🏠 Create, edit, and delete property listings  
- 🖼️ Upload images for each listing  
- 📍 Add location for each property  
- 💰 Set price per night  
- 🔍 View all listings in a clean UI  
- 📱 Responsive design (mobile + desktop)  
- 🗺️ Map integration for location viewing  
- 🔐 User authentication system  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, EJS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB + Mongoose  
- **Image Storage:** Cloudinary / Local Storage  
- **Deployment:** Render  

---

## 📁 Project Structure

models/ → Database schemas  
routes/ → Express routes  
controllers/ → Business logic  
views/ → EJS templates  
public/ → CSS, JS, images  
middleware/ → Authentication & validation  

---

## ⚙️ Installation & Setup

### 1. Clone repository
git clone https://github.com/unlimitedcode07/paradise.git


## 2. Install dependencies
npm install

## 3. Create .env file
MONGO_URL=your_mongodb_connection
CLOUDINARY_NAME=your_cloud_name
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret

## 4. Run project
npm start

## 6. Open in browser
http://localhost:3000
