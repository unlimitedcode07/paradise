# 🏡 Paradise Valley - Airbnb Clone

A full-stack web application inspired by Airbnb where users can explore, create, edit, and book beautiful stay listings.

🌐 Live Demo: https://paradise-valley.onrender.com/listing  
💻 GitHub Repo: https://github.com/unlimitedcode07/paradise  

---

## 🚀 Features

- 🏠 Create, edit, and delete property listings
- 🖼️ Upload images for each listing
- 📍 Add location and description
- 💰 Set price per night
- 🔍 View all listings in a clean UI
- 📱 Responsive design (mobile + desktop)
- 🗺️ Map integration (location-based viewing)
- 🔐 User authentication (if implemented)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap / EJS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Image Storage:** Cloudinary / Local storage
- **Deployment:** Render

---

## 📁 Project Structure


models/ - Database schemas
routes/ - Express routes
controllers/ - Logic for routes
views/ - EJS templates
public/ - CSS, JS, assets
middleware/ - Auth & validation

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/unlimitedcode07/paradise.git

Go to project folder:

cd paradise

Install dependencies:

npm install

Create .env file:

MONGO_URL=your_mongodb_connection
CLOUDINARY_NAME=your_cloud_name
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret

Run the project:

npm start

Open in browser:
